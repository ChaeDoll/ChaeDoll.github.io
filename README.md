# 임채윤 | Personal Website

XR/통신/AI 연구자의 개인 GitHub Pages

## 🚀 사용 방법

1. 이 레포지토리를 GitHub에 업로드
2. Repository Settings > Pages에서 GitHub Pages 활성화
3. [index.html](index.html)의 연락처 정보 수정

## ✏️ 수정이 필요한 부분

### index.html
- GitHub 링크: `yourusername`을 본인의 GitHub 사용자명으로 변경
- 이메일: `your.email@example.com`을 본인 이메일로 변경
- LinkedIn: `yourprofile`을 본인 프로필로 변경

## 📁 파일 구조

```
├── index.html      # 메인 페이지
├── styles.css      # 스타일시트
└── README.md       # 설명 문서
```

## 🎨 커스터마이징

- 색상 변경: [styles.css](styles.css)의 `#667eea`, `#764ba2` 값 수정
- 내용 추가: [index.html](index.html)에 새로운 섹션 추가
- 프로필 사진: `<header>` 섹션에 이미지 추가 가능
